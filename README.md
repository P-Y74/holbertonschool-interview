# holbertonschool_interview

## Description

This repository contains algorithmic and technical interview practice projects completed as part of the Holberton School curriculum.

It includes exercises focused on problem solving, data structures, and implementation logic commonly encountered in coding interviews.  
Each project directory contains one specific challenge with its own source files and, when relevant, a dedicated `README.md`.

This repository is intended to grow over time as more interview-style problems are completed.

## Educational Context

This work was completed as part of the Holberton School curriculum.  
The goal of this repository is to strengthen problem-solving skills, improve understanding of core data structures, and prepare for technical interviews through practical exercises.

## Repository Structure

The repository is organized by project.

### [heap_insert](./heap_insert)
Implementation of insertion in a Max Binary Heap.

Key concepts:
- binary trees
- heaps
- tree traversal
- pointer manipulation

---

### [insert_in_sorted_linked_list](./insert_in_sorted_linked_list)
Insertion of a node into a sorted singly linked list while preserving order.

Key concepts:
- linked lists
- sorted insertion
- pointer handling
- edge case management

---

### [lockboxes](./lockboxes)
Problem-solving exercise involving graph-like traversal and reachability.

Key concepts:
- lists
- traversal logic
- visited state tracking
- algorithmic reasoning

## Learning Objectives

Through these projects, this repository helps develop:
